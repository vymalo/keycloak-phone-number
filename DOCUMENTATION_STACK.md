# Documentation Stack

This document provides an overview of the documentation stack for the Keycloak Phone Number Login Plugin.

## Primary Documentation

*   **[`README.md`](README.md)**: This file serves as the main source of documentation for the project. It provides a comprehensive overview of the plugin's functionality, architecture, and setup instructions for Docker and Kubernetes environments. It also includes a compatibility table for different Keycloak versions, a list of implemented providers, and a detailed list of environment variables for configuration.

## API Documentation

*   **[`openapi/sms-open-api.yml`](openapi/sms-open-api.yml)**: This file provides the OpenAPI specification for the SMS service used by the plugin. It defines the API endpoints for sending and validating SMS messages, including the request and response schemas. It also specifies the security schemes supported by the API.

## Changelog

*   **[`CHANGELOG.md`](CHANGELOG.md)**: This file is intended to track the project's version history. It currently contains a limited number of entries, and could be improved with more detailed and consistent formatting.

## Visuals

*   **[`docs/screenshot.png`](docs/screenshot.png)**: This image file in the `docs/` directory provides a visual example of the plugin's user interface, showing the screen where users are prompted to enter their phone number.
